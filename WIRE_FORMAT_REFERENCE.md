# Wire Format Reference – v3 Block Settings

Authoritative reference for how each block type's UI settings map to the v3 API wire format.
Derived from `journey-logic-commons/src/schemas/v4/blocks/` transform handlers.

## Common Settings (all blocks)

Every block's `uischema.options` can include:

| Setting | Wire field | Notes |
|---------|-----------|-------|
| `showPaper` | `options.showPaper` | Boolean – card background |
| `title` | `options.title` | Block title/header text |
| `subtitle` | `options.subtitle` | Block subtitle |
| `variablePath` | `options.variablePath` | Auto-generated by `createStep()` |
| `stickyOnMobile` | `options.stickyOnMobile` | Sticky positioning on mobile |
| `stickyOnMobileIndex` | `options.stickyOnMobileIndex` | Sticky z-index |
| `halfWidth` | `options.halfWidth` | Half-width in group layouts |

---

## Cross-cutting Patterns

### 1. Parallel Arrays (choices)
Single Choice, Multiple Choice, and Hyperlink blocks use parallel arrays:
- `options.options[]` – values
- `options.optionsLabels[]` – display labels
- `options.optionsIcons[]` – icon objects `{ name }` (optional)
- `options.optionsImages[]` – image objects `{ id?, url, dimension? }` (optional)
- `options.optionsImageDescriptions[]` – alt text (optional)

### 2. Dispatcher: `Control` type
The generic `Control` type dispatches based on the JSON schema:
- **Boolean schema** → Binary Input (toggle/checkbox)
- **String schema with `enum`** → Single Choice
- **String schema without `enum`** → Text Input

### 3. Block References
Cross-block references use `"stepIndex/blockName"` string paths.

### 4. Field Display Pattern
Composite blocks (PI, Contact, Account, Address) use:
- Field **exists** in `options.fields` → `display: true`
- Field has `required: true` → `required: true`
- Field **absent** → `display: false`

## Block Types

### Text Input
**Control:** `Control` (schema type `string`, no `enum`)

| UI Setting | Wire Format |
|-----------|------------|
| Label | `label` (top-level, not in options) |
| Placeholder | `schema.properties[name].examples[0]` |
| Multiline | `options.multiline: true` |
| Required | `options.required: true` + `schema.required[]` |
| Format validation | `options.regex` pattern |

### Single Choice
**Control:** `Control` (schema type `string` with `enum`)

| UI Setting | Wire Format |
|-----------|------------|
| Button mode | `options.button: true` |
| Radio mode | `options.radio: true` |
| Image mode | `options.imageButton: true` |
| Dropdown mode | none of the above flags |
| Button layout | `options.buttonType: 'row' \| 'stack' \| 'chip'` |
| Image layout | `options.imageType: 'infographic' \| 'icon'` |
| Choice values | `options.options: string[]` |
| Choice labels | `options.optionsLabels: string[]` |
| Choice icons | `options.optionsIcons: Array<{ name } \| undefined>` |
| Choice images | `options.optionsImages: Array<{ url, dimension? }>` |
| Dropdown from file | `options.buttonOptionsFileId: string` + `options.autocomplete: true` |
| Required | `options.required: true` |

**Schema:** `{ type: 'string', enum: [values] }`

### Multiple Choice
**Control:** `MultichoiceControl`

Same parallel arrays as Single Choice, plus:

| UI Setting | Wire Format |
|-----------|------------|
| Checkbox mode | `options.uiType: 'checkbox'` (default) |
| Button mode | `options.uiType: 'button'` |
| Image mode | `options.uiType: 'imageButton'` |
| Max selection | `options.maxSelection: number \| 'Infinity'` |

**Schema:** `{ type: 'array' }`

### Binary Input
**Control:** `Control` (schema type `boolean`)

| UI Setting | Wire Format |
|-----------|------------|
| Switch mode | `options.toggle: true` |
| Checkbox mode | `options.toggle: false` or absent |
| Default value | `schema.properties[name].default: boolean` |
| Binary marker | `options._binaryInput: true` (SDK factory marker) |

### Number Input
**Control:** `NumberInputControl`

| UI Setting | Wire Format |
|-----------|------------|
| Unit display | `options.fields.numberInput.unit: { show, label }` |
| Format | `options.fields.numberInput.format` |
| Range | `options.fields.numberInput.range: { min, max }` |
| Suggestions | `options.suggestions: Array<{ label, value }>` |
| Suggestion icon | `options.iconInputName` |
| Adornment icon | `options.input_icon` |
| Mode | `options.mode` |
| Formula | `options.formula` |
| Label | `options.fields.numberInput.label` |

### Date Picker
**Control:** `DatePickerControl`

| UI Setting | Wire Format |
|-----------|------------|
| Start date field | `options.fields.startDate: { label?, limits?: { disablePast } }` |
| End date field | `options.fields.endDate: { label?, limits? }` |
| Show time | `options.showTime: boolean` |
| Time intervals | `options.timeIntervals` |
| Disable days | `options.disableDays: number[]` (weekday indices) |

### Personal Information
**Control:** `PersonalInformationControl`

| UI Setting | Wire Format |
|-----------|------------|
| Private mode | `options.customerType: 'PRIVATE'` or absent |
| Business mode | `options.customerType: 'BUSINESS'` |
| User-defined mode | `options.customerType: 'BOTH'` |
| Purpose labels | `options.purposeLabels: string[]` |
| Block title | `options.title: string` |
| Fields | `options.fields: { salutation?, title?, firstName?, lastName?, email?, telephone?, birthDate?, companyName?, registryCourt?, registerNumber? }` |
| Field required | `options.fields.firstName: { required: true }` |
| Field choices | `options.fields.salutation: { options: [...] }` |

### Contact
**Control:** `ContactControl`

| UI Setting | Wire Format |
|-----------|------------|
| Purpose | `options.purpose: string[]` (mapped from `purposeLabels`) |

| Entity prefill | `options.entityPrefill: { mode }` |
| Fields | `options.fields: { salutation?, title?, firstName?, lastName?, email?, confirmationEmail?, telephone?, birthDate? }` |
| Autofill from PI | `options.isAutoFillPIEnabled: true` + `options.autoFillPISettings: { relatedBlock: "stepId/blockName" }` |

### Account

### Address
**Control:** `AddressControl`

| UI Setting | Wire Format |
|-----------|------------|
| Country | `options.countryAddressSettings.countryCode: string` |
| Autocomplete | `options.countryAddressSettings.enableAutoComplete: boolean` |
| Free text | `options.countryAddressSettings.enableFreeText: boolean` |
| Google Maps | `options.googleMapsIntegrationOptions: { isGoogleMapsEnabled, isRepositioningAllowed }` |

### Product Selection
**Control:** `ProductSelectionControl`

| UI Setting | Wire Format |
|-----------|------------|
| Catalog | `options.catalog: 'epilot'` (default) |
| External catalog | `options.externalIntegration: { enabled: true, integrationId }` |
| Products (static) | `options.products: [{ productId, priceId, isFeatured?, featuredLabel?, featuredColor?, featuredLabelColor? }]` |
| Products type | `options.productSelectionConfiguratorType: 'product_selector' \| 'cross_sell_product_selector'` |
| Selection | `options.selectionType: 'many'` (multi) or absent/other (single) |
| Show quantity | `options.showQuantity: boolean` |
| Show images | `options.showImages: boolean` |
| Show price components | `options.displayPriceComponents: boolean` |
| CTA text | `options.ctaTextOption` |
| Features limit | `options.featuresLimit: number` |
| Trailing zeros | `options.showTrailingDecimalZeros: boolean` |
| Recurring total | `options.displayRecurringTotal: boolean` + `options.recurringTotalUnitOption` |
| Layout variant | `options.tileLayoutVariant: { variant, options[] }` |
| Alignment | `options.justifyContent: 'flex-start' \| 'space-between' \| 'center'` |
| Variable pricing | `options.blockMappings: [{ type: 'pricing', stepId, blockName, metadata: { productId, price: { priceId, componentId } } }]` |
| Filters | `options.filters: [{ productId, priceId }]` |
